# Jeu de données réaliste pour tester Hub-Lib

Ce fichier propose des ressources publiques et pérennes (repos GitHub, documentations officielles, jeux de données libres, etc.) correspondant aux **8 catégories prédéfinies** de Hub-Lib. Chaque entrée inclut les champs principaux du modèle `Resource` pour un import manuel (interface "Créer une ressource" ou script Prisma). Les URL pointent vers des contenus stables et librement accessibles.

> Champs recommandés pour l'import : `title`, `description`, `category`, `tags`, `resourceType`, `externalUrl` (ou `githubUrl`), `language`, `license`.

## Suggestions d'utilisation
- Saisir manuellement quelques entrées dans l'UI "Créer une ressource" en conservant la catégorie exacte (ex. `Templates & Modèles`).
- Pour un script d'initialisation Prisma, mapper les champs ci-dessus vers le modèle `Resource` et utiliser `resourceType` = `external_link` ou `github_repo` selon le cas.
- Ajouter des `tags` en minuscules afin de profiter des filtres (ex. `["docker", "devops", "compose"]`).
- Les licences indiquées permettent de laisser ces ressources en production sans contrainte majeure.
